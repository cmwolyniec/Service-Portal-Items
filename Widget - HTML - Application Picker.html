The Disaster Recovery Dashboard provides visibility into Disaster Recovery readiness for critical applications and services as identified by the Premera Business Continuity Team. To begin, select an item from the drop-down list below. The page is then divided into three sections:

  • Server Replication Status: Displays associated servers and related replication health (NOTE: currently only displays virtual server replication health.)
  • Most recently recorded Disaster Recovery plan and exercise dates
  • Application Support contact information
